Spotify Collab - easily manage collaborative playlists.

Create and share collaborative playlists with friends.

Each collaborator on a playlist can add songs to a separate list of songs, which are voted on by each collaborator. Once the yes votes reach a certain percentage for a specific song in the list, that song is added to the main collaborative playlist.


  1. Create a collaborative playlist
  2. Invite friends to collaborate
  3. Both you and your collaborators can add songs to a separate list of songs to be "voted" on
  4. Once a song reaches a certain percentage of yes votes, the song is added to the collaborative playlist

Login to Spotify and start collaborating!
